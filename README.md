# Всем привет.

Я буду делать проект для облегчения подсчета материалов в Genshin Impact.
Будет веб-сайтик с базой данных. Для реализации буду использовать Django, Python, SQLite.
## Основные идеи что будет делать программа:
1. "добавить своего персонажа" - указать, что у тебя есть определённый персонаж из базы, указать его уровень и таланты.
2. "прокачать персонажа" - повысить уровень персонажа, или уровень талантов персонажа (если ты прокачал его в игре)
3. "посчитать материалы" - подсчет материалов, необходимых для прокачки ВСЕХ персонажей до максимального уровня 
(в идеале максимальный уровень задаем в настройках - потому что в геншине нет смысла прям макс макс качать). на этой странице желательны фильтры по типу материала, а также сортировка по региону (по умолчанию), по возрастанию и убыванию необходимого количества

## админские функции 
(если не воруем данные напрямую у хоеверс или где-то еще) 

1. добавить материалы в базу (появился новый моб или босс в игре)
2. добавить персонажа (появился новый в игре)
